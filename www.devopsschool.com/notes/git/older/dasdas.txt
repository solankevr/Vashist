How to start working with git?
1. Create a repo
> git init

2. adding a files
> git add filename

3. Set the user.name & user.email
> git config user.name "Rajesh"
> git config user.email "rajesh@scmgalaxy.com"

4. commit a file
> git commit -m"This is my first commit"
