Hardware

Prerequisite
Every machine should have 3 VM's with Redhat 6/Centos version.
Each VM should be able to connect with Internet and Internal corporate proxy should be set accordingly.
Every machine should minimum 2GB Ram
Every Base Machine (Windows Machine) should have Putty, and WinScp
Every Base Machine (Windows Machine) should have at least 8GB RAM
Every VM should have fully qualified hostname which should be pinged from another machine.
Every VM should have static IP setup
All the VM should ping each other!!
Each Machine should have “sudo yum update” run in full cycle.
Corporate Proxy server:port should be available to use by participants in case of download limitation and firewall.
All the VM should ping each other!!
