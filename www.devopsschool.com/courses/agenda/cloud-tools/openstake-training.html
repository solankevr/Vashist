About

OpenStack Compute Essentials
Days - 2
This course is a two-day intro class covering the core OpenStack compute projects including Neutron, Nova, Glance, Keystone and Horizon. This is the same material covered in the first two days of our OpenStack Fundamentals class with topics that range from understanding the architecture, to installing and configuring. This course is a great primer for anyone interested in OpenStack Compute. This course is designed for technical individuals and consists of lectures, discussions, demos and hands on labs.

OpenStack Object Storage Essentials
Days - 2
This course is a two-day intro class covering the OpenStack Object Storage project, codenamed Swift. This is the same material covered in the last two days of our OpenStack Fundamentals class wtih topics that include understanding the Swift components, architecture, installation and maintenance. This course is a great primer for anyone interested in OpenStack Object Storage. This course is designed for technical individuals with minimum Swift experience. The course consists of lectures, discussions, demos and hands on labs.
