About

Bitbucket is the Git solution for professional teams. Bitbucket is a web-based hosting service for projects that use either the Mercurial (since launch) or Git (since October 2011) revision control systems. Bitbucket offers both commercial plans and free accounts. Distributed version control system that makes it easy for you to collaborate with your team. The only collaborative Git solution that massively scales.

Built for professional teams

Flexible deployment models
